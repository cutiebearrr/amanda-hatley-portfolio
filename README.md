# Amanda Hatley Portfolio

This is a static portfolio site.

## Run locally

Open `index.html` in a browser.

## Deploy

Upload the contents of this folder to any static host such as Netlify, Vercel, GitHub Pages, Cloudflare Pages, or a regular web server.

The main page is `index.html`.

## Notes

The page uses Google Fonts from `fonts.googleapis.com`. Everything else needed for the page is contained in `index.html`.
